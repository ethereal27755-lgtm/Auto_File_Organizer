# Auto File Organizer

A simple Python script that automatically organizes files in a folder by type.

## Features
- Automatically sorts files into categories (Documents, Images, Videos, etc.)
- Prevents file overwrite by renaming duplicates
- Easy to use

## Usage
1. Modify the folder path in the script
2. Run the script:

py auto_file_organizer.py

## Example
Before:
Downloads/
    a.pdf
    b.jpg
    c.txt

After:
Downloads/
    Documents/a.pdf
    Images/b.jpg
    Texts/c.txt
